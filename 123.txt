Базовая система
    • base, base-devel — минимальный базовый комплект и инструменты для разработки.
    • linux, linux-headers — ядро и заголовки для совместимости с драйверами.
Микрокод процессора
    • amd-ucode — микрокод для AMD Ryzen 7 7700 (вместо intel-ucode).
Драйверы графики NVIDIA
    • nvidia, nvidia-utils, lib32-nvidia-utils — драйверы NVIDIA GTX 980 Ti и поддержка 32-битных приложений.
    • nvidia-settings — графический интерфейс для настройки видеокарты.
Сетевое управление
    • networkmanager, iwd, wpa_supplicant — управление сетью и Wi-Fi.
    • networkmanager-applet — графическое приложение для управления сетью.
Звуковая система
    • pipewire, wireplumber, alsa-utils — современная аудиоплатформа.
    • pipewire-pulse, pipewire-alsa — обратная совместимость с PulseAudio и ALSA.
Гибридная графика и Wayland
    • wayland — протокол для графического окружения.
    • xdg-utils, xdg-desktop-portal, xdg-desktop-portal-wlr — интеграция с окружением.
Hyprland и утилиты
    • hyprland — гибридный тайловый Wayland-композитор.
    • waybar — панель задач для Hyprland.
    • wofi — запускатель приложений.
    • wl-clipboard — управление буфером обмена.
    • grim, slurp — захват скриншотов и выделение областей.
Дополнительные компоненты
    • kitty — быстрый современный терминал.
    • mako — демон уведомлений.
    • polkit, polkit-gnome — управление правами доступа.
    • light — управление яркостью экрана.

Конфигурация для гибридной графики AMD+NVIDIA
1. Параметры ядра (обязательно)
Отредактируйте /etc/default/grub:
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash nvidia_drm.modeset=1"

Обновите конфигурацию:
sudo grub-mkconfig -o /boot/grub/grub.cfg

2. Добавление NVIDIA модулей в initramfs
Отредактируйте /etc/mkinitcpio.conf:
MODULES=(nvidia nvidia_modeset nvidia_uvm nvidia_drm)

Пересоберите initramfs:
sudo mkinitcpio -P

3. Конфигурация Hyprland для гибридной графики
В файле ~/.config/hypr/hyprland.conf добавьте переменные окружения:
env = AQ_DRM_DEVICES,/dev/dri/card0:/dev/dri/card1

Для AMD GPU (основной рендерер) используйте более приоритетный card:
env = AQ_DRM_DEVICES,/dev/dri/cardX:/dev/dri/cardY

где cardX — AMD GPU, cardY — NVIDIA GPU. Проверьте карты через ls /dev/dri/ и lspci.
4. Дополнительные переменные для оптимизации
# Отключение аппаратных курсоров (может потребоваться для NVIDIA)
env = WLR_NO_HARDWARE_CURSORS,1

# Для правильной работы Wayland на NVIDIA
env = LIBVA_DRIVER_NAME,nvidia
env = __GLX_VENDOR_LIBRARY_NAME,nvidia
env = GBM_BACKEND,nvidia-drm
